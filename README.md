RESTORING DEGRADED AFRICAN LANDSCAPES (ReDEAL) PROJECT

Metadata for spatial, field and household survey data collected for the implementation of workpackages 1, 2 and 3. 

The project was implemented in two sites located in Kuresoi and Nyando areas in West Kenya.
#############################################################################################################################################################
#############################################################################################################################################################
Field sites

This relates to information about sites that were selected for installation of exclosures, monitoring of livestock grazing behavior and soil sampling

File location (or link): "\ERU_consent.xlsx"

The main variables are:
Plot ID: An integer that is unique for each site
Latitude: in decimal degrees
Longitude: in decimal degrees
Location: Name of town where site is located
Sublocation: Lower categorisation of location within Kenyan's administrative classification system
ERU: Confirmed ERU classification after remote sensing analysis and categorisation
Previous land use: Previous land use before conversion to grassland
Age_land use: Number of years since conversion to grassland
Soil type: Dominant soil type (0-30cm)
Elevation: Elevation above mean level
